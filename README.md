# DSA Visualization

A Python-based GUI application that visually demonstrates various **Sorting and Searching Algorithms** using the Tkinter library.

This project was built to make understanding Data Structures and Algorithms more interactive through real-time animations, complexity analysis, and user controls.

---

## Features

### Sorting Algorithms

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

### Searching Algorithms

* Linear Search
* Binary Search
* Jump Search
* Interpolation Search
* Exponential Search

---

## Visualization Features

* Real-time algorithm animation
* Pause and Resume execution
* Reset with new randomized datasets
* Adjustable animation speed
* Operation counter:

  * Swaps for sorting algorithms
  * Comparisons for searching algorithms
* Execution timer
* Target input for searching algorithms
* Visual highlighting of:

  * Current element under inspection
  * Eliminated elements
  * Successfully found elements
* Time and space complexity information for every algorithm

## Installation

1. Clone the repository:

```
git clone <repository-link>
```

2. Navigate to the project directory:

```
cd DSA-Visualization
```

3. Run the application:

```
python DSA_Visualization.py
```
